## Getting Started

### Prerequisites

- Node.js 16+ 
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Build

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```
